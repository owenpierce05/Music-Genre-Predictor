# Spotify Genre Classifier
A machine learning project investigating whether a song's genre can be predicted from its Spotify audio features alone.

Dataset: Spotify Tracks Dataset containing 114,000 tracks across 125 genres (Kaggle) [https://www.kaggle.com/datasets/maharshipandya/-spotify-tracks-dataset]

Key result: XGBoost classifier achieved 73% accuracy across 8 genres

Tools: Pandas, scikit-learn, XGBoost, matplotlib, seaborn
